# 给Cosmos贡献代码

## 贡献方法

如果你在使用过程中发现了问题，或者有更好的改进，请积极的为我们贡献代码，主要的贡献途径包含以下几种：

### 框架基础及模板

可以直接向 http://gitlab.sftcwl.com/fe-treasury/cosmos.git 仓库提起merge request，如若没有权限，请直接联系前端团队负责人。

| 开发者                  | 介绍                                               |
| -------------          | --------------------------------------------------|
|  黄斌旭  |    ICON管理方案                                         |
|  何波  |    项目文档化                                       |
|  韩秀敏  |    国际化方案                                       |
|  杨扬  |    异步封装方案                                       |
|  李淳  |    权限管理方案                                       |
|  何波、魏延  |    公共管理方案                                       |

### npm包

为了方便迭代管理，我们把一部分功能抽成了npm包，如若有对应方面的修改，请联系对应的开发管理员；

| 包名              | 介绍                                | 负责人          |
| -------------          | --------------------------------------------------| -------------------|
| sofa-scripts     | 核心开发工具&框架管理                                | 张宁         |
| cosmos-builder-webpack     | 编译、打包相关                                | 尚万郑         |
| eslint-config-sf-cosmos     | Eslint规则配置                                | 任亚楠         |
| cosmos-utils     | 公共方法                                | 王承乾         |
| eden-remote-deploy     | 免密远程代码部署                                | 李淳         |
| longan     | 埋点及错误监控                               | 张驰阳         |
| sofa-map     | 对高德地图的React封装                               | 杨扬         |
| easy-print     | 面单打印助手                               | 董方旭         |
